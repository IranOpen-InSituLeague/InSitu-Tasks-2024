# InSitu-Tasks-2024
تسک‌های لیگ طراحی و ساخت InSitu در سال ۲۰۲۴ شامل هفت راند تخصصی است: تعقیب خط، ماز، بازو، آتش‌نشان، کنترلی، ماشین خودران و راند سوپرتیم روبوبال، به‌همراه راند ربات ورزشکار. این راندها با هدف ارزیابی توان طراحی، برنامه‌نویسی، کنترل و خلاقیت تیم‌ها در چالش‌های رباتیکی برگزار می‌شوند.
